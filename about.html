What is GDSC??

Google Developer Student Clubs adalah sebuah komunitas yang dibuat oleh Google dan ditujukan bagi para mahasiswa yang tertarik dengan perkembangan teknologi Google.

Google Developer Student Clubs Universitas Andalas (GDSC Unand) terbuka bagi mereka yang tertarik untuk belajar bersama melalui video bersama, diskusi terbuka atau berkumpul bersama dengan beragam topik.

Divisi GDSC

Front End Developer

Bertanggung jawab mengimplementasikan desain dan konsep yang ada di web dengan menggunakan beberapa bahasa pemrograman seperti HTML, CSS, dan JavaScript menggunakan teknologi React dan Firebase

Back End Developer

Bertanggung jawab mengembangkan keterampilan dan skill dari sisi server sebuah sistem, mempelajari teknologi seperti Node.js, Express.js, MongoDB, dan MySQL untuk mengelola data.

Machine Learning

Melakukan penelitian, pengembangan, desain kecerdasan buatan hingga bertanggung jawab untuk pembelajaran mesin, serta mempelajari bahasa Python menggunakan tools dari Google seperti Google Collab, Tensorflow, dan masih banyak lagi.

Mobile Developer

Spesialiasi dalam teknologi mobile. Hal itu meliputi membuat aplikasi untuk Android Google dengan mempelajari Dart Programming. Framework, maupun Firebase.

UI/UX

Bertanggung jawab merancang antarmuka pengguna yang menarik dan fungsional, serta menciptakan pengalaman pengguna yang memuaskan melalui tools Figma&fFigjam serta properti Auto layout.

Human Resource

Bertanggung jawab untuk mengelola sumber daya GDSC UNAND. Melakukan penilaian keterampilan sumber daya GDSC UNAND, serta menciptakan membentuk keakraban anggota komunitas.

Curriculum

Bertanggung jawab untuk menyusun materi pembelajaran selama satu periode GDSC UNAND. Serta mengatur timeline studyjam dan batasan dari setiap divisi technical.

Event Organizer

Bertanggung jawab atas penjadwalan, perencanaan, dan pengelolaan event GDSC UNAND hingga rancangan kegiatan yang dilakukan oleh GDSC UNAND.

Public Relations

Bertanggung jawab sebagai narahubung GDSC UNAND dengan pihak eksternal, serta bertanggung jawab dalam hal membangun kerjasama, serta menjaga nama baik GDSC UNAND.

Graphic Design

Bertanggung jawab untuk membuat design konten kreatif serta aset graphic yang dimiliki GDSC UNAND. Melakukan dokumentasi acara, konten, dan digital branding GDSC UNAND.

Content Writing

Bertanggung jawab untuk menyusun rencana konten setiap sosial media, copywriting. menganalisis kemajuan setiap sosial media, pembuatan caption, serta mengelola sosial media dengan bijak.
